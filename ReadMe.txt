
Description:
This program was designed to use object inheritance to create a bank account system.
The sytem reads in instructions from BankTransIn.Txt and parses them into different transaction
objects. These transactions include deposits, withdrawls, transfers between accounts,
opening new acounts.

Instructions:
The program can be run in from windows command prompt using Bank.exe
